1.  Introduction

   An implementation MUST preserve the message identifier while a
   message is retransmitted.

2.  Message Format

   The following type carries a message.

   ExampleMessage ::= SEQUENCE {
       messageId   INTEGER,
       payload     OCTET STRING
   }

2.1.  Encoding

   message = message-id SP payload

   The payload is encoded after the message identifier.
